# 🎬 Cómo nombrar los videos de ejercicios

Nombra cada archivo EXACTAMENTE con el ID de variante + `.mp4` (ej. `press-horizontal-barra.mp4`).
Así el sistema lo ubica solo. Organizado por músculo para que sepas qué es cada uno.

> Si grabaste una variante que NO está aquí, mándamela y la agrego al banco.
